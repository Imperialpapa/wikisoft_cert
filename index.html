01 — PROJECT OVERVIEW
프로젝트 개요
K-IFRS 1019호는 종업원에게 제공하는 모든 형태의 급여에 대한 회계처리를 규정합니다.

왜 이 프로젝트가 필요한가?

K-IFRS 1019호에 따른 종업원급여 회계는 보험수리적 가정, 할인율 산정, 확정급여채무(DBO) 계산 등 복잡한 과정을 수반합니다. 기존에는 수작업에 의존하여 오류 발생 가능성이 높고, 감사 대응에 많은 시간이 소요되었습니다.

본 시스템은 종업원급여 관련 재무 데이터를 자동으로 수집 및 분석하여, 기준서 요구사항과의 적합성을 실시간으로 검증합니다.

validation_engine.py
# K-IFRS 1019 검증 엔진 코어
class KIFRSValidator:
  def validate_dbo(self, data):
    """확정급여채무 검증"""
    result = self.check_actuarial(data)
    return result.is_valid
VALIDATION SCOPE
  • ▸확정급여채무(DBO) 산출의 보험수리적 가정 적정성 검증
  • ▸사외적립자산의 공정가치 측정 및 수익률 검증
  • ▸할인율 적용의 일관성 및 시장 데이터 정합성 확인
  • ▸재측정요소의 기타포괄손익 반영 적정성 검증
  • ▸과거근무원가 및 축소/정산 손익 인식 검증
  • ▸공시 요구사항 충족 여부 자동 점검
  • ▸전기 대비 변동 분석 및 이상치 탐지
